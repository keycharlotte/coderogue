# CodeRogue

一个基于Godot 4.4和C#开发的Roguelike游戏项目。

## 项目结构
CodeRogue/
├── Scenes/                    # 场景文件
│   ├── Main/                  # 主场景
│   ├── Player/                # 玩家相关场景
│   ├── Enemy/                 # 敌人场景
│   ├── UI/                    # UI场景
│   └── Level/                 # 关卡场景
├── Scripts/                   # C#脚本
│   ├── Core/                  # 核心系统
│   ├── Player/                # 玩家系统
│   ├── Enemy/                 # 敌人系统
│   ├── Combat/                # 战斗系统
│   ├── UI/                    # UI系统
│   ├── Level/                 # 关卡系统
│   ├── Data/                  # 数据类
│   ├── Utils/                 # 工具类
│   ├── Components/            # 组件系统
│   └── Editor/                # 编辑器工具
├── Assets/                    # 资源文件
├── Resources/                 # Godot资源
├── Design/                    # 设计文档
└── Tools/                     # 开发工具
## 开发环境

- Godot 4.4+
- .NET 6.0+
- C# 10+

## 核心特性

- 模块化架构设计
- 组件化系统
- AI友好的代码结构
- 完整的游戏状态管理
- 可扩展的敌人AI系统

## 开始开发

1. 在Godot中打开项目
2. 确保启用了C#支持
3. 构建项目以生成C#程序集
4. 开始编码！
